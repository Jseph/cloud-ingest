On-Premises Agent Setup Instructions:

  1. One-time gcloud setup:
    • Download and install gcloud if it's not already on your system.
    • Run
      gcloud auth application-default login
       
  2. Download and extract the on-prem agent: Linux (64-bit)
  3. In a shell window navigate to the path where you extracted the agent.
  4. Run
    ./agent --pubsub-prefix={{ PUBSUB_PREFIX }} --projectid={{ projectId }}

    To specify an agent log directory other than your system's default temp directory, add -log_dir <dir>
