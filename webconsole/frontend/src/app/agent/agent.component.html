On-Prem Agent Setup Instructions:

  1. Download and extract the on-prem agent: Linux (64-bit)
  2. In a shell window navigate to the path where you extracted the agent.
  3. Run
    ./agent -projectid {{ projectId }}

    To specify an agent log directory other than your system's default temp directory, add -log_dir <dir>
