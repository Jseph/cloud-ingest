Instructions to download the control plane logs:

  1. Open a new tab, and navigate to console.cloud.google.com.
  2. On the left navigation sidebar, click on Compute Engine.
  3. Locate the cloud-ingest-dcp vm instance.
  4. Click the SSH button under the 'Connect' column.
  5. On the browser window that opens, run:
    docker images --filter=reference='gcr.io/mbassiouny-test/cloud-ingest' --format "ancestor={{.ID}}" | xargs docker ps --format "{{.ID}}" --filter | xargs docker logs > dcp.log 2>&1
    This will create a dcp.log file in the current directory with the logs. You can view your logs by running less dcp.log
  6. To download your logs:
    1. Run readlink -f dcp.log to get the absolute path to the dcp.log file.
    2. Click the tool icon on the top right corner of your vm window. Select 'Download file'. Copy and paste the absolute path to dcp.log that you obtained above.
